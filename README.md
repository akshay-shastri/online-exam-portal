# 🎓 Smart Exam Portal

A modern AI-powered Online Examination & Proctoring System built using **React.js**, **Spring Boot**, and **MySQL**. The platform provides a secure, scalable, and user-friendly environment for conducting online examinations with real-time monitoring, candidate management, and performance analytics.

## 🌐 Live Demo

**Frontend:** https://online-exam-portal-rho-ten.vercel.app

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/akshay-shastri/online-exam-portal.git
```

---

## Frontend

```bash
cd frontend
npm install
npm run dev
```

---

## Backend

```bash
cd backend
mvn spring-boot:run
```
